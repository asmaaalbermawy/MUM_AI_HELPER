# MUM AI Helper 🤱

An AI-powered assistant web app designed to support moms with parenting guidance, built with **Next.js**, **TypeScript**, and **Tailwind CSS**.


## Features

- Chat with an AI assistant for parenting questions and support
- Clean, responsive interface for mobile and desktop

## Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS

## Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/asmaaalbermawy/MUM_AI_HELPER.git
cd MUM_AI_HELPER
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
